# AI-Solutions Website — Complete Build (22 Files)

## Quick Start (5 minutes)
1. Extract this ZIP to: C:\xampp\htdocs\
   (creates C:\xampp\htdocs\ai-solutions\)
2. Open terminal in ai-solutions folder, run: composer require mongodb/mongodb
3. Start Apache + MongoDB in XAMPP
4. Visit: http://localhost/ai-solutions/setup_db.php (run ONCE, then delete the file)
5. Visit: http://localhost/ai-solutions/
